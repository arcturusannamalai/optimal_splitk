Welcome to doe_gensplit!
========================
**doe_gensplit** is a Python library designed for generalizing split-plots. It includes the fast update procedures
outlined in the article. It generalizes the update formula's to models of the type :math:`Y = X \beta + \sum_{i=1}^k Z_i \gamma_i`.

The design section details how the code is designed, for mathematical derivations, see the article and supplement. The API
section refers to exact documentation per function when using the library.
